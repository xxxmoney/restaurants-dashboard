
# Proxy

## Description
 - A simple proxy made for this project in php
 - Uses chrome driver and chrome browser to get html content of pages

## Getting started
1. Install the required dependencies:
   ```bash
   composer install
   ```
   
2. Extract the ChromeDriver and Chrome binary to the `bin` directory


3. Run (you can use package.json to run it from root):
   ```bash
   npm run dev:proxy
   ```
